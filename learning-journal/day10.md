## What is an LLM??
LLM or "Large Language Model" is an AI model trained on massive dataset to predict next word or sentence by understanding human language sementics.

## What is a Prompt?
Prompt is nothing but instruction which is given to LLM to produce desired response.

## Why is Prompt Engineering important?
LLM response is as good as your prompt. So crafting a good prompt is like engineering task hence called prompt engineering.

## How does an LLM API differ from a normal REST API?
LLM api is also a rest api that interects with LLM. So LLM api is rest api but not all rest APIs are LLM APIs. Reason being :
- Rest API generates stored response while LLM API creates response
- LLM API exposes predefined method to interact and get data while LLM API return data based on prompt given
