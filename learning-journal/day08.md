## What is a module?
Reusable single python file containig common functions that can be used in by other python files

## What is a package?
Folder containing related python files containng specific functional area or business space, like employee package can have Employee, Manager, HR classes and give bonus common function.

## Why use imports?
To use module and packages in other python file


## Why use virtual environments.
To logically segregate one project and its dependencies with aother to avoid version conflicts.

## How would I organize a large AI project?
Break reusable codes in modules and packages. Have different packages for agents, LLM, RAG etc.
